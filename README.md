# Jeffrey Cleveland - Personal Website

Personal website and portfolio for Jeffrey Cleveland, economist.

## Live Site

Visit: [jeffreycleveland.github.io](https://jeffreycleveland.github.io)

## Structure

- `index.html` - Home page
- `charts.html` - Economic charts collection
- `css/main.css` - Main stylesheet
- `fonts/` - Custom font files for the website
- `PDFcharts/` - High-resolution PDF versions of economic data visualizations

## Development

This is a static site hosted on GitHub Pages. Simply push changes to the main branch to deploy.
